# Learning_Python_Day6

Today's topic is lists!

**Goal:** Store and manipulate collections of data using Python lists

- **task1:** Create a list of 5 favorite foods and print each one

- **task2:** Add an item to the list using append()

- **task3:** Remove an item using remove()

- **task4:** Access and print the first, last, and middle items

- **task5:** Loop through the list and print each item with its number

- **task6:** Create a list of numbers and calculate the sum and average

**mini-project:**
Shopping List Manager:  
Create an interactive program with a menu:

1. Add item  
2. Remove item  
3. View list  
4. Clear all  
5. Exit  

Use a while loop to keep the menu running and a list to store items.  
The program should continue until the user chooses Exit.
